# Repository Operating Contract

## VCS Policy

- Primary platform: GitHub
- GitHub is the canonical remote for this repository.
- Prefer pull request workflow over direct integration.
- Do not treat GitLab as the primary or default hosting target for this repo.

## Publication

- Default public hosting target: `github.com`
- Prefer `gh` and the GitHub app tooling for repository and pull request operations.

## Safety Boundary

- Do not run destructive git commands unless explicitly requested.
- Do not push directly to protected branches.
